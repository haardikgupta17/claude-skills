# claude-skills

A collection of custom skills (slash commands) for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [sprint-pilot](./sprint-pilot/) | AI-powered sprint co-pilot — Jira ticket analysis, velocity tracking, scope creep detection, and sprint reviews |

## What are Claude Code skills?

Skills are markdown files that give Claude Code specialized capabilities via slash commands. Drop a `.md` file into `~/.claude/commands/` and it becomes a `/command` you can invoke in any Claude Code session.

## Installation

```bash
git clone https://github.com/haardikgupta17/claude-skills.git
```

Then follow the setup instructions in each skill's folder.
